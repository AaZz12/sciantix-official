Solver for the spatially averaged solution of the PDE [dy/dt = D div grad y + S - L y] We apply a spectral approach in space, projecting the equation on the eigenfunctions of the laplacian operator. We use the first order backward Euler solver in time. The number of terms in the expansion, N, is fixed a priori.
